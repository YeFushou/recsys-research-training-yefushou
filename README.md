推荐系统研究训练记录
本仓库用于记录推荐系统研究训练过程中的学习计划、每日进度、课程作业和知识笔记。内容围绕机器学习、深度学习、PyTorch、推荐系统基础与论文阅读逐步沉淀，方便后续复盘、补全和扩展。

仓库目标
建立推荐系统研究所需的基础知识体系。
持续记录每日学习进度、问题和阶段性总结。
整理课程作业、知识点笔记和可复用学习材料。
为后续论文阅读、模型复现和实验实践提供索引。
目录结构
.
├── assignment/        # 课程作业
├── daily_logs/        # 每日学习记录
├── knowledge_base/    # 主题知识库
├── learning_plan.md   # 学习计划
└── README.md          # 项目说明
当前内容
assignment/：课程作业记录，目前包含第一次课作业。
daily_logs/：按日期记录学习过程，目前包含 2026-07-10 和 2026-07-11 的日志。
knowledge_base/：按主题整理知识点，包括数学、Python、PyTorch、机器学习、深度学习、推荐系统、论文阅读和工具使用。
learning_plan.md：学习计划入口，用于持续补充阶段目标和时间安排。
学习主线
补齐 Python、数学和机器学习基础。
学习深度学习基础概念与 PyTorch 实践。
系统整理推荐系统核心算法和常见任务。
开展论文阅读，记录问题、方法、实验和结论。
结合课程作业和实验复现形成阶段性总结。
记录规范
每日记录放在 daily_logs/YYYY-MM-DD.md。
课程作业放在 assignment/，文件名尽量体现课程序号和主题。
主题笔记放在 knowledge_base/，一个主题对应一个 Markdown 文件。
新增图片或资料时，优先使用仓库内的相对路径，避免依赖本地绝对路径。
