# 推荐系统研究训练记录

本仓库用于记录我的推荐系统（Recommendation System）学习与研究训练过程。

主要记录假期学习计划、每日学习日志、课程作业、知识库整理以及论文阅读过程。

学习内容围绕：

- Python 编程基础
- 数学基础与机器学习方法
- 推荐系统算法
- 深度学习推荐模型
- 论文阅读与实验复现


---

## 仓库目标

- 建立推荐系统研究所需的基础知识体系。
- 形成长期积累的学习记录和研究资料。
- 记录学习过程中的问题、思考和阶段性总结。
- 为后续推荐系统实验、论文阅读和模型复现提供资料支持。
