# 八、问题及排查过程


## 问题1：PyTorch导入失败


错误：

```
WinError 127

c10_cuda.dll加载失败
```



## 原因分析


原环境使用了CUDA版本PyTorch。

但是当前电脑CUDA环境不匹配，导致动态库加载失败。



## 解决方法


重新创建实验环境：

```bash
conda create -n sasrec python=3.10
```


安装CPU版本PyTorch。


重新运行：

```bash
python sasrec_demo.py --device cpu --epochs 1
```



最终成功完成实验。
